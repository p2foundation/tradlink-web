# Download Hero Video

## Quick Download Guide

The hero section needs a video file to display a background video. Follow these steps:

### Option 1: Mixkit (Recommended - Free & High Quality)

1. **Visit**: https://mixkit.co/free-stock-video/cargo-ship-full-of-containers-4011/
2. **Click** the "Download Free" button
3. **Save** the file as: `hero-video.mp4` in this folder (`web/public/videos/`)
4. The video will automatically be used once saved!

### Option 2: Coverr (Alternative)

1. **Visit**: https://coverr.co/videos/shipping-containers
2. **Click** the download button
3. **Rename** the file to: `hero-video.mp4`
4. **Save** it in this folder (`web/public/videos/`)

### Option 3: Pixabay

1. **Visit**: https://pixabay.com/videos/search/shipping%20containers/
2. **Choose** a video and download
3. **Rename** to: `hero-video.mp4`
4. **Save** in this folder (`web/public/videos/`)

## File Location

The video file should be saved as:
```
web/public/videos/hero-video.mp4
```

## After Download

Once you've saved the video file, the hero section will automatically use it. No code changes needed!

The video will:
- ✅ Auto-play in the background
- ✅ Loop continuously
- ✅ Be muted (as required by browsers)
- ✅ Fall back to a static image if it fails to load

## Video Requirements

- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Duration**: Any length (will loop)
- **File Size**: Try to keep under 10MB for faster loading
